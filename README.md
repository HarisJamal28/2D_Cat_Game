# Madzy's Little Adventure

## About the Game
**Madzy's Little Adventure** is a fun, fast-paced, never-ending arcade game built with Phaser 3. Players control Madzy, a spunky cat, on her mission to stomp out pesky male cat suitors who have been constantly bothering her.  

The objective is simple: jump, stomp, and survive while avoiding damage from enemies. As the game progresses, it becomes more challenging and action-packed.

## Gameplay
- **Control Madzy** using arrow keys (or W/A/D) to move and jump.  
- **Stomp enemies** from above to defeat them and earn points.  
- **Avoid taking damage** from enemies; losing all hearts ends the game.  
- **Score points** for each enemy stomp, with bonus rewards for hitting score milestones.  

*Future updates will include collectible items to restore health and special power-ups to enhance gameplay.*

## Features
- Smooth player movement and jump mechanics.  
- Enemy AI with walking and jumping behaviors.  
- Health system represented by hearts.  
- Dynamic scoring and fun messages for milestone achievements.  
- Pause and restart functionality.  
- **Future plans:** HTML-based title screen for easier customization, collectible items, and power-ups.
